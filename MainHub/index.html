      Builder's Cheat Sheet

      Use these exact tags in your text file to generate your world!

      # Level Name

      Creates a new Map Node. All standard text written below this header becomes the "Reading Material" for the level.

      - [ ] Task Name

      Creates a Quest (Checklist item). Checking these off rewards you with coins!

      !FLASH! Question :: Answer

      Generates a Spaced Repetition flashcard. Remember to keep the spaces around the :: separator!

      !QUIZZ! Question :: Answer

      Creates a Quiz question. You must score 100% on the Quiz to unlock the Exam.

      !EXAM! Question :: Answer

      Creates a Final Exam question. Passing this turns your map node green!

      !GAME! Name of Game

      Assigns a Mini-Game to the section level. Available games: Arcane Defense, Trivia, Memory, Flash-Match, Spellweaver, Ritual Alignment, Cloze Trial, True/False Blitz, Glimpse & Recall.

      Pro Tips 💡

      • Every 5th section automatically becomes a Diamond Milestone, and the very last section always becomes the Crown Boss Node.
      • To turn a map node completely Gold, you must pass the Exam and successfully burn all the flashcards inside that section!

      ☁️ Sync Cloud Save

      Export your library data to your computer. Move the downloaded file into your MainHub folder and Publish to GitHub to instantly update your phone!

      How Save Sync Works

      1) On PC: click Export PC Save to Cloud, then place the downloaded saveData.json inside MainHub and publish.

      2) On iPhone: open the app and use Force Sync Phone From Cloud when you want the latest worlds.

      3) Your phone progress is merged and preserved when possible (cards, tasks, section progress), not blindly overwritten.

      4) No saveData.json yet? That is fine. Start creating worlds normally, then export your first save file from this menu.

      Privacy note: saveData.json is visible to anyone with access to where you publish it. Use a private repo/host if you want personal study data hidden from others.
